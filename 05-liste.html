Les listes HTML

Les listes Ordonnées

Une liste ordonnée se définit avec la balise <ol></ol>

Les entrées de la liste sont définit avec la balise <li></li>

exemple:
  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Les listes Non Ordonnées

Une liste non-ordonnée se définit avec la balise <ul></ul>

Les entrées de la liste sont définit avec la balise <li></li>

exemple:
  • Item 1
  • Item 2
  • Item 3
  • Item 4

Les listes de définition

Une liste de definition se définit avec la balise <dl></dl>

exemple:
HTML
HyperText Markup Language
CSS
Cascading Style Sheet
PHP
PHP Hypertext Processor
